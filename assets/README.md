# assets

Đặt logo ứng dụng tại đây với tên chính xác:

    assets/logo.png

Ảnh nên là PNG vuông, nền trong suốt (khuyến nghị ≥ 1024×1024).
Logo dùng cho: màn hình splash khi mở app, native splash, và icon ứng dụng khi cài đặt.

Sau khi thêm `logo.png`, chạy:

    flutter pub get
    dart run flutter_launcher_icons
    dart run flutter_native_splash:create
